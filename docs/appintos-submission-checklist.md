# 앱인토스(iOS) 출시 제출 체크리스트

## 1. 빌드/패키징
- [ ] `npm install`
- [ ] `npm run build --workspace app`
- [ ] `app/dist` 산출물 확인 (HTML/CSS/JS 로딩 에러 없음)
- [ ] 앱인토스 iOS 웹뷰 템플릿에 `app/dist` 탑재
- [ ] iPhone 실기기 실행 확인(첫 화면 진입, 터치 입력, 백그라운드 복귀)

## 2. 기능 검수
- [ ] 12슬롯(3x4) 모두 심기/성장/수확 동작
- [ ] 오프라인 진행 30분/2시간/8시간/12시간(8시간 캡) 검증
- [ ] 기기 시간 과거 이동 시 오프라인 진행 차단 확인
- [ ] 식물 12종 첫 수확 시 도감 등록 확인
- [ ] 화분 9종/배경 4종 구매 및 적용 확인
- [ ] 로컬 저장/복구(강제 종료 후 상태 유지) 확인

## 3. 스토어 메타데이터
- [ ] 앱 이름(국문), 서브타이틀(국문)
- [ ] 설명(국문), 키워드(국문)
- [ ] 스크린샷 세트(iPhone)
- [ ] 연령 등급/콘텐츠 등급 입력
- [ ] 개인정보 처리방침 URL 입력
- [ ] 심사 노트 입력

## 4. 심사 노트 핵심 문구
- HTML5 게임을 웹뷰로 제공
- 계정 로그인/결제 기능 없음
- 광고 SDK 미탑재(v1)
- 로컬 저장 기반 싱글 플레이 게임

## 5. 제출 전 최종 확인
- [ ] 앱 크래시 없음
- [ ] 오프라인 상태에서 실행 가능
- [ ] 텍스트 오탈자 검수 완료
- [ ] 버전/빌드 번호 정합성 확인
