## 3. 현재 구현 상태 (코드 기준)
- 메인 씬: `app/src/scenes/GardenGameScene.ts` (약 10k 라인)

### 홈 탭
- 심기/성장/수확 루프 동작
- 개화/수확 시 시각 연출 강화
- 상단 리소스 UI 정렬 복구
- 출석 오버레이 진입/수령 연동

### 정원 탭
- 카툰 모드 고정
- 2-3-2 계단형 슬롯 배치
- 슬롯 탭 → 편집 모달
- 편집 모달에서 식물/화분 변경 후 저장
- 빈 슬롯 가이드 텍스트 표시
- `비우기` 모드로 슬롯 즉시 회수 가능

### 도감 탭
- 카테고리: `꽃 / 식물 / 세트보상`
- 세트보상 카드 및 수령 기능
- 수령 가능 빨간점(하단 탭/카드 버튼) 표시

### 상점/출석
- 상점 기본 구매/업그레이드 루프 동작
- 7일 출석 + 7일 해금 보상 동작
- 광고 SDK는 아직 미연동
