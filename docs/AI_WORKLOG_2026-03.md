# AI 월간 작업로그 (2026-03)

## 2026-03-13

### 작업 범위
- 실제 사용자 플레이 테스트 전환(테스트 강제값 제거)
- 정원 탭 A/B/C열 슬롯 위치 미세조정
- 도감 세트카드 `필요` 문구 가독성 고정 포맷 적용

### 주요 구현
- 테스트 강제 상태 초기화
  - 테스트 강제 코인/해금 코드 제거
    - `FORCE_SET_TEST_COINS`, `FORCE_UNLOCK_ALL_SPECIES_FOR_TEST`, `TEST_COIN_AMOUNT` 삭제
  - 도감 임시 해금(히아신스 강제 발견) 제거
  - 정원 2페이지 관엽/3페이지 다육 강제 프리셋 배치 제거
  - 신규 기본 저장 데이터의 `discoveredSpeciesIds`를 빈 배열로 복구
  - 로컬 저장 키를 `haetae.idle-garden.save.v2`로 상향해 기존 테스트 세이브와 분리
- 정원 탭 위치 조정
  - 슬롯 렌더 오프셋 기준:
    - A열(Back): `-10px`
    - B열(Mid): `+15px`
    - C열(Front): `+50px`
  - 슬롯 시각요소/이름표/터치영역이 함께 이동되도록 동일 오프셋 적용 유지
- 정원 편집 모달 문구 정리
  - 저장 버튼 위 `변경 없음` 사유 문구 제거
  - 저장 불가 사유는 토스트/버튼 상태로만 안내
- 도감 세트카드 문구 포맷 개선
  - `필요 :` 문구를 전체 필요 종 목록으로 표시
  - 1줄 3종 고정 + 나머지 종을 다음 줄로 강제 줄바꿈
    - 예: `필요 : 튤립, 장미, 데이지,` 다음 줄 `아네모네, 금잔화, 동백꽃`

### 문서 업데이트
- `AI_HANDOFF.md`: 최신 상태(플레이 테스트 기준)로 스냅샷 갱신
- `AI_TASK_BOARD.md`: `RELEASE-001` 상태 갱신
- `KNOWN_ISSUES.md`: 테스트 플래그 이슈 상태 갱신
