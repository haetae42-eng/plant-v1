# 회귀 QA 체크리스트 (v1)

## 정원(Home)
- [ ] 슬롯 선택/해제 동작
- [ ] 씨앗 심기 시 재고 감소
- [ ] 성장 시간 카운트다운 정상
- [ ] 개화 후 수확 가능 상태 전환
- [ ] 수확 시 코인 증가, 슬롯 비움
- [ ] 화분 변경(보유 화분 순환)

## 정원(Decorate)
- [ ] 슬롯 탭 시 편집 모달 오픈
- [ ] 편집 모달 식물 `<`,`>` 변경 동작
- [ ] 편집 모달 화분 `<`,`>` 변경 동작
- [ ] 저장 가능 상태에서만 `저장` 버튼 활성화
- [ ] 저장 불가 상태면 버튼 위 이유 문구 표시(예: `변경 없음`)
- [ ] 저장 성공 시 `저장 완료` 토스트
- [ ] 비우기 ON 상태에서 슬롯 탭 시 즉시 회수
- [ ] 비우기 ON/OFF 토스트 문구 정상(`비우기 ON`, `비우기 OFF`)
- [ ] 탭 전환 후 정원 복귀 시 모달/선택 상태 꼬임 없음
- [ ] 페이지 전환(`이전/다음`) 후 슬롯 데이터 유지

## 도감(Collection)
- [ ] 미발견/발견 카드 구분
- [ ] 첫 수확 시 자동 등록
- [ ] 페이지 이동(이전/다음) 정상

## 상점(Shop)
- [ ] 씨앗 구매 코인 차감/재고 증가
- [ ] 화분 구매 후 보유 처리
- [ ] 배경 구매 후 적용 가능
- [ ] 코인 부족 시 구매 차단

## 설정(Settings)
- [ ] 효과음/BGM 토글
- [ ] 저장 즉시 실행
- [ ] 초기화 2회 확인 동작

## 출석(7일 해금)
- [ ] DAY 카드 해금 라벨/아이콘/기본 보상 텍스트 겹침 없음
- [ ] 오늘(또는 다음) 보상/해금 요약 2줄이 하단에 정상 표시
- [ ] 오늘 출석 수령 시 기본 보상 + 해금 보상 동시 지급
- [ ] 출석 수령 토스트가 짧은 메시지로 출력
- [ ] 7일차 수령 후 루프가 1일차로 정상 초기화
