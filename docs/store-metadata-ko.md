# 스토어 메타데이터(국문) 초안

## 앱 이름
나만의 정원: 방치형 식물 키우기

## 서브타이틀
식물 수집과 꾸미기를 동시에

## 설명
심고, 기다리고, 수확하세요.
나만의 속도로 식물을 키우는 세로형 방치 게임입니다.

- 12종의 식물을 수집하고 도감을 완성
- 9종 화분과 4종 배경으로 정원을 꾸미기
- 앱을 꺼둔 동안에도 최대 8시간 오프라인 성장
- 간단한 조작으로 한 손 플레이 가능

정원은 로컬 저장으로 안전하게 유지되며,
첫 수확한 식물은 도감에 자동 등록됩니다.

## 키워드
방치형,식물키우기,정원꾸미기,수집게임,힐링게임,세로게임

## 심사 노트 예시
이 앱은 HTML5 기반 게임을 iOS 웹뷰 컨테이너에서 실행합니다.
로그인/결제 기능은 포함하지 않았고, 로컬 저장으로 진행 데이터를 유지합니다.
광고 SDK는 현재 버전에 포함되지 않습니다.
